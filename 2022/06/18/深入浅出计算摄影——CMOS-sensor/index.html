CMOS sensor 物理结构

sensor stack


CMOS sensor stack



CMOS sensor stack

- image sensor主要分为CMOS sensor和CCD sensor。二者的区别主要有 - CCD sensor有更大的单pixel感光面积 - CMOS和CCCD的pixel array阵列排列和readout方式不同 - CMOS更经济实惠 ## sensor floorplan

sensor floorplan


  • optical black
    • 用一个不透光物体完全挡住sensor，该区域捕捉到的信号强度，即为黑电平
  • dummy border
    • 不用的像素，猜测主要是sensor制造工艺的考虑，需要进行数据宽度对齐
  • readout
    • 电信号读出
  • 单个像素可以建模为一个反向偏置的二极管+一个COMS电容。
